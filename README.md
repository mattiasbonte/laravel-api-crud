# Basic CRUD API Resource Generator

### These 2 scripts helps with generating or removing basic CRUD functionality (inside a docker container)

## **Tech**
- Bash
- Laravel 8
- Docker

## **crud.sh**
#### Generates:
- Model
- CRUD Controller & basic methods
- Policy & basic methods
- Resource (id & name)
- Request (id & name)
- Routing


## **remove.sh**
#### Removes:
- Model
- Controller
- Policy
- Resource
- Request
- Routing
